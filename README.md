# Job Portal Web Application

A dynamic Job Portal Web Application built using Laravel, PHP, and MySQL.  
The platform allows job seekers to apply for jobs and recruiters/admins to manage job postings efficiently.

---

##  Features

### User (Job Seeker)
- User Registration & Login
- Profile Creation & Update
- Browse & Search Jobs
- Apply for Jobs
- View Applied Jobs History

###  Admin
- Admin Dashboard
- Manage Job Categories
- Add / Edit / Delete Jobs
- View Job Applications
- Select Candidates

###  Authentication & Security
- Role-based Access Control (Admin, User)
- Secure Login System
- Protected Routes
- Environment-based configuration

---

##  Tech Stack

- **Backend:** Laravel (PHP Framework)
- **Frontend:** Blade Templates, HTML, CSS, Bootstrap
- **Database:** MySQL
- **Authentication:** Laravel Auth
- **Server:** XAMPP / Apache
